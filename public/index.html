I/O is expensive

  • L1: 3 cycles
  • L2: 14 cycles
  • RAM: 250 cycles
  • DISK: 41,000,000 cycles
  • NETWORK: 240,000,000 cycles

dealing with I/O

  • synchronous
  • fork a new process
  • threads
  • using event loop
